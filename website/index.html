How it works

✂️

Copy as usual

Smart Clipboard sits in your tray and captures every copy in the background — no hotkey to remember, no workflow to change. A small HUD confirms each capture.

🏷️

Everything gets classified

Colors, URLs, emails, code snippets, file paths, currency, dates — each capture is typed automatically and embedded for semantic search.

⌨️

Search with Alt+Space

Keyword or natural language — full-text and vector search merged with a recency boost. "That auth snippet from yesterday" just works.

Private by design

A clipboard manager sees everything you copy. That's exactly why this one is built to never leak.

🔒

100% offline

Capture, classification, embedding, and search all run locally. No account, no cloud, no telemetry.

🚫

Password managers ignored

Conceal-format copies are dropped before they're ever stored. KeePass, 1Password, and Bitwarden are excluded out of the box.

🛡️

Secrets never indexed

API keys, tokens, JWTs, and high-entropy strings are detected and kept out of search indexes and embeddings entirely.

🔑

Encrypted at rest

SQLCipher database with the key stored in Windows Credential Manager — the file on disk is opaque ciphertext.
